# LearnEng

写一个帮助学习英语的小程序

## 运行说明

以LearnEng文件夹作为工程目录，在编译器中打开

## 主要文件说明

* data：用于数据爬取与保存
  * download_ebooks：下载电子书
  * download_qsbdc_phrases：下载英文词组
* utils文件夹：用于存放各个模块可能使用的公共函数
  * translate：翻译单词或词组
  * web：发送网络请求
  * log：用于记录日志
